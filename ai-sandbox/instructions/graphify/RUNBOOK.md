# Graphify Runbook

Graphify is a derived AST/graph index. It is not the source of truth.

## Rules

- Run commands from the workspace root.
- Parse only the configured project root from `ai-sandbox/config/project.yaml`.
- Keep generated graph artifacts under `ai-sandbox/graph/`.
- Do not parse `ai-sandbox/` as project source.
- Run `python3 ai-sandbox/scripts/audit_sandbox.py` after graph refresh.

## Standard Refresh

```bash
python3 ai-sandbox/scripts/refresh_graph.py
python3 ai-sandbox/scripts/audit_sandbox.py
```

## Source Status

Graph relationships generated by adapters are deterministic helper facts. Label uncertain relationships as `inferred` or `needs_verification`.
