# LOGBOOKS.md

Sandbox changes go to `ai-sandbox/logbooks/LOGBOOK_SANDBOX_IA.md`.

Project code, behavior, architecture, build, deployment, or documentation changes go to `ai-sandbox/logbooks/LOGBOOK_PROJECT.md`.

Each entry must include:

- request;
- context read;
- work performed;
- verification commands and result;
- files changed;
- residual risks or skipped checks.
